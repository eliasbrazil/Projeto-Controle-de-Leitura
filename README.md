# Projeto-Controle-de-Leitura
Controle de leitura

O projeto tem por finalidade de efetuar um controle simples de leituras de livros.

Entradas:
        - Data
        - Cadastro Livro
            - Titulo do Livro
            - Quantidades de páginas
            - Autor
        - Livro
        - Paginas lidas
  
Efetuar o código nas seguintes linguagens de programação:
  - SQL
  - Python
  - Java
  - HTML
  - C++
  - PHP
  - Android
